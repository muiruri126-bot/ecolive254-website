# ECoLIve Kenya — Website

Marketing website for **Ecosystem Conservation 4 Livelihood Improvement (ECoLIve)** — a Kenyan non-profit ending the poverty cycle through nature-based solutions and climate-smart agriculture.

Static site (HTML / CSS / vanilla JS) deployed via **GitHub Pages**.

- **Live site:** https://www.ecolive254.org/
- **Content & branding** sourced from the official ECoLIve organization profile and newsletter.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Single-page site (hero, about, programs, impact, gallery, contact) |
| `styles.css` | Brand styling — green / brown / white palette from the ECoLIve logo |
| `script.js` | Mobile nav, scroll reveal, gallery lightbox, contact form |
| `assets/` | Logo + field photos |
| `CNAME` | Custom domain for GitHub Pages |
| `.nojekyll` | Tells Pages to serve files verbatim (no Jekyll build) |

## Local preview

Open `index.html` in a browser, or serve the folder with any static server.
